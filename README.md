## Zip/MRPack import
### Prism
1. Simply copy the link to the download of the .zip version of the latest release.

2. Create a new instance, click "import" on the left, and paste it into the box.

## Installing directly into Prism
1. Make a new instance for 1.21.1 and make sure you select NeoForge as a modloader.

![Guide image 1](/guide0.png)

![Guide image 2](/guide1.png)

2. Once it's done, right click the instance and select "edit."

![Guide image 3](/guide2.png)

3. Next, copy everything from the `mods` folder in this repo into an instance's mods folder (which you can access with a button at the bottom right of its mods tab).

![Guide image 4](/guide3.png)

![Guide image 5](/guide4.png)

You can also install any recommended optional client-side mods into the same area.
